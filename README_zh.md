## Windows Microsoft Store 最新版下载安装脚本

支持 arm arm64 x86 x64单架构安装,支持以下版本的安装和更新
- Windows LTSC 2021
- Windows LTSC 2019
- Windows 1809 及以上版本

---

## 下载和运行

1. 无需Python环境：

下载并运行 https://github.com/Goojoe/LTSC-ADD-Microsoft-Store/releases

3. 需要手动安装Python环境：

```
pip install -r requirements.txt
python main.py
```
